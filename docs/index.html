Overview

Abstract

Vertebral Heart Score (VHS) estimation is a clinically grounded measurement task in veterinary cardiology: experts identify anatomical landmarks, compute geometric axes, and apply fixed diagnostic thresholds. Existing automated methods are often evaluated using aggregate accuracy or regression error alone, which can hide clinically important failures near VHS decision boundaries.

We present CardioMark, a benchmark for automated VHS estimation built from 21,465 annotated lateral thoracic canine radiographs from 12,385 dogs. Each image includes six cardiac and vertebral keypoints, a continuous VHS measurement, and threshold-based diagnostic labels. The dataset was constructed using a dedicated web-based annotation tool that supports rapid keypoint placement, real-time VHS computation, model-assisted review, and quality control.

CardioMark evaluates the full measurement pipeline: landmark localization, geometric consistency, continuous VHS estimation, threshold-based classification, subgroup reliability, and decision instability near diagnostic boundaries. Our analysis shows that models with strong aggregate metrics can still fail near clinical thresholds, motivating evaluation protocols aligned with how VHS is actually used in practice.

Annotation Tool

VHSMarker / VetMark Annotation Platform

A web-based interface for rapid, standardized cardiac keypoint annotation and real-time VHS calculation.

Tool Interface

Interactive interface for six anatomical keypoints, real-time VHS display, model-assisted prediction review, and quality-control actions.

Technical Architecture
  • Backend: Flask/Python with OpenCV image processing.
  • Frontend: HTML5 Canvas and JavaScript for responsive annotation.
  • Real-time API: immediate VHS computation after keypoint updates.
  • ML integration: predict button for automated model output display.
  • Quality control: mark invalid cases and compare predicted landmarks.
  • Viewing controls: zooming, panning, and brightness/contrast adjustment.
Core Functionality

The platform supports precise labeling of six anatomical keypoints: cardiac apex, tracheal bifurcation, short-axis endpoints, and vertebral reference points. It computes VHS in real time using:

VHS = 6 × (AB + CD) / EF

Here, AB is the long cardiac axis, CD is the short cardiac axis, and EF is the vertebral reference length. The tool was designed to reduce annotation time to approximately 10–12 seconds per image while maintaining a standardized measurement workflow.

Evaluation Design

Benchmark Protocol

CardioMark evaluates the complete measurement process instead of only final class accuracy.

1
Landmark Localization

Evaluate six cardiac and vertebral keypoints using coordinate-based errors and normalized localization metrics.

2
Geometric Measurement

Compute VHS from predicted geometry and assess alignment with the clinical measurement pipeline.

3
Threshold-Based Diagnosis

Classify VHS into Normal, Borderline, and Enlarged categories using fixed clinical thresholds at 8.2 and 10.0 VU.

4
Boundary and Subgroup Reliability

Analyze near-threshold accuracy, perturbation-induced flip rate, class-wise performance, and worst-group reliability.

Evaluation Pipeline

Predicted landmarks are converted into geometric axes, continuous VHS values, threshold-based labels, and boundary-sensitive reliability metrics.

Takeaway

Conclusion

CardioMark provides a standardized benchmark for automated VHS estimation that connects dataset construction, annotation tooling, geometric measurement, and clinically meaningful evaluation. By releasing structured keypoint annotations and evaluation protocols, the benchmark enables researchers to compare models across the complete VHS pipeline rather than relying only on aggregate classification accuracy.

The central finding is that VHS-based diagnosis is highly sensitive near fixed clinical thresholds. Strong overall performance does not guarantee reliable behavior for near-boundary cases or under small geometric perturbations. CardioMark therefore emphasizes boundary sensitivity, subgroup reliability, and decision instability as essential components of evaluation.
